# Upgrade or Uninstall

### Homebrew

```ruby
brew upgrade warrensbox/tap/hubapp
```
### Linux

Rerun:

```sh
curl -L https://raw.githubusercontent.com/warrensbox/hubapp/release/install.sh | bash
```

## Uninstall:

### Homebrew

```ruby
brew uninstall warrensbox/tap/hubapp
```
### Linux

Rerun:

```sh
rm /usr/local/bin/hubapp
```
